持久加密USB安装

  • LUKS：Linux Unified Key Setup

    • 磁盘分区加密规范
    • 不依赖于操作系统的磁盘级加密

  • 将镜像刻录到U盘

    • dd if=kali-linux-1.1.0-amd64.iso of=/dev/sdb bs=1M # bs代表每次复制的大小

  • 为U盘创建额外分区

    • parted 磁盘分区工具

      print devices
print
mkpart primary start end
quit

  • 使用LUKS加密分区

    • 前端 cryptsetup

      cryptsetup --verbose --verify-passphrase luksFormat /dev/sdb3
...

  • 打开加密分区

    cryptysetup luksOpen /dev/sdb3 usb

  • 格式化加密分区+卷标

    mkfs.ext4 /dev/mapper/usb
e2label /dev/mapper/usb persistence

  • 挂载加密分区

    mount /dev/mapper/usb /mnt/usb/
echo "/ union" > /mnt/usb/persistence.conf

  • 结束操作

    umount /dev/mapper/usb
cryptsetup luksClose /dev/mapper/usb

熟悉BASH命令

ls  # 列出当前目录 -l 详细 -a 列出隐藏文件 -h human  --sort=size/time  以size or time排序
cd  # 更改目录
pwd  # 查看当前目录
cat  # 显示文件内容
more  # 逐屏显示文本内容  similar command: less
tail  # 默认显示最下面10行  -number 显示最下面 number lines 
    watch -n 2 tail -20 /var/log/message
cp  # copy files  -r copy directory
rm  # delete files
top  # 查看系统进程或者杀死进程  键入k，输入PID
ps  # -ef 查看详细进程信息  -aux 
grep  # 筛选输出内容 grep "string" file
egrep  # grep升级版  -v 排除某些字符串
awk  # 对输出内容分块，默认为空格  awk '{print $5}'  输出第五块的内容 -F:  选择以:为间隔分割字符
cut  # 对内容分块  -d ':' 指定分块的间隔符  -f 1  显示第一个分块
sort  # 排序
uniq  # 去重
seq 1 3 # 序列
ifconfig  # ifconfig eth0 down / up
macchanger  # -m 00:11:11:11:11:11 eth0  修改网卡mac地址
netstat  # -pantu  查看网络连接信息
route  # 添加，删除路由信息
mount  # mount -o loop kali.iso /media/cdrom
    # -o options 主要用来描述设备或档案的挂接方式。
    # loop：用来把一个文件当成硬盘分区挂接上系统 
    # ro：采用只读方式挂接设备 
    # rw：采用读写方式挂接设备 
    # iocharset：指定访问文件系统所用字符集
dmesg  # 直接显示日志信息
find  # find / -name nmap  查找nmap   -iname 忽略大小写  -exec 执行其他bash命令
whereis  # 在数据库中查找 -b 查找二进制文件
    # 先使用updatedb更新数据库
echo  # 输出  -n 不换行输出
vi / vim  # 文本编辑
mtr IP  # Mytraceroute  查看当前路由信息
dhclient eth0   # 强制使网卡重新获得IP地址
dpkg -L firece   # 列出与某个程序有关的所有目录

  • /etc/resolv.conf 当前计算机的dns配置文件

    netstat -pantu | egrep -v '0.0.0.0|:::' | awk '{print $5}' | egrep -v 'and|Address' | cut -d ':' -f 1 | sort | uniq > ip
